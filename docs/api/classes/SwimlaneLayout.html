Class SwimlaneLayout

A hierarchical layout algorithm.

Constructor: mxSwimlaneLayout

Constructs a new hierarchical layout algorithm.

Arguments:

graph - Reference to the enclosing Graph. orientation - Optional constant that defines the orientation of this layout. deterministic - Optional boolean that specifies if this layout should be deterministic. Default is true.
